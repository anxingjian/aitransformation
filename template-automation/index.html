The Problem

We spend most of our time executing repetitive tasks instead of making design decisions.

01

Weak template and prompt choices from PMs

PM-provided templates and prompts are not ideal, requiring designers to spend additional time correcting both.
02

Thumbnail generation process is very inefficient

Upload → wait → review → repeat. The thumbnail generation process is very mechanical and time-consuming.
03

Extra manual step to test visual quality in Figma

Every image must be manually placed into Figma's card template — just to see "does this look good at 110px?"
"What if AI could accelerate the entire workflow — discover trends, generate templates, and evaluate quality?"
Solution · Part 1

AI-Powered Market Trends Discovery

Using data-driven trend analysis to guide content direction

Trend Scraping + Import
  • Crawl competitor platforms for trending topics and high-engagement prompts
  • Identify which styles and subjects drive real user interest
  • Output structured template suggestions with quality prompts
  • Export as JSON → one-click import into the generation tool
Live Demo
Solution · Part 2

Batch Generation + AI Evaluation

Replace manual cycles with parallel automation — then let AI judge the results.

  • Import JSON → templates auto-created with prompts and rules
  • Upload before images → one-click batch generation
  • Real-time progress streaming in the browser
  • AI evaluation auto-triggered after generation
  • Preview before/after at actual card size in tool
  • AI recommends best cover 💡 and flags issues ⚠️
